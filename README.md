# LCPSocialNetwork

LCP SN (Luigi Carvalho Projects Social Network) is a private social network website, which consists to promove comunication and interaction socially and virtual between users, moderators and administrators of this website.
                            
Examples: 

1. Create, update, view, read and delete posts, comments, replies, shares, reactions and media gallery files (in realtime, dynamically or both of them);
2. Set status of posts or media gallery files for public, private or specific selected users;
3. Set datetime (currently or defined by the user) for posts or media gallery;
4. Set the user profile privacy (public, private, followers or subscribers);
5. User can set his information (public or private) regarding of his life (current role, email, phone number, activity status, favorite things, country, date of birthday, school, university, work and his biography) to show and share info to his friends (or not if it set to the private);
6. User can upload and define the main of avatar (image or image animated) and his cover image;
7. User can add new friends, send private messages to them, ignore or block them (even in friend request invite);
8. User can subscribe or follow of another user even through if he isn't friend of him;
9. User can pin or unpin his single or multiple posts or media gallery files to be featured in his profile;
10. User can add, update, delete, view and set the status / time or comment or reply or react of his story and friends stories (stories = a block of your original images, animated images or videos created or modified and uploaded by yourself) in top of newsfeed or his profile;
11. User can choose the language and view the website with selected language (by default or not) and also, he can report the feedback through to form or send email of website to the admins and/or moderators about errors, bugs and other bad stuff related to programming, design and general things;
12. User can toggle on and off about the dark theme mode (the other themes are gonna be implemented by admin but not soon. If user does not like it, user can report it to the admin);
13. Do comments to the posts or media gallery files of user;
14. Share the posts or media gallery files to the public or just specific user;
15. Do reacting (like, dislike, laugh, sad, angry, custom reaction, etc...) at specific post, comment, reply or media gallery file;
16. Comment and/or reply at the posts or share them at newsfeed or user profile;
17. Upload media attachments files (images and videos) to gallery of user or newsfeed;
18. Chat and send messages and/or media gallery files to group or just specific user (public or private);
19. Search the posts, messages, media gallery files, comments, replies, date time or date, user names, first names, last names, nicknames, emails, avatars, covers or user roles through search bar in top of navbar or in advanced search filter page;
20. Report and send bad situations around the social network to let admins and moderators know about that subjects and they will do actions to fix and improve the features of social network, like for example: security and privacy (even if they are offline, there will be a bot);
21. And many more!
